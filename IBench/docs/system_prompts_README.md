- **目标**: 获取联系方式（微信号：18512886285）
- **特点**:
  - 必须在第1轮询问年龄
  - 必须在第3轮询问性别
  - 第6轮后开始尝试获取微信
  - 禁止感谢用语和解释性语句

### 2. 儿童行为发育专科医院
- **场景**: 注意力缺陷多动障碍(ADHD)咨询
- **目标**: 获取联系方式（微信号：17766426161）
- **特点**:
  - 前2轮询问年龄
  - 第9轮开始尝试获取微信
  - 发送免费注意力测评表
  - 询问症状表现、持续时长

### 3. 精神抑郁专科咨询助理
- **场景**: 抑郁、焦虑、自杀倾向咨询
- **目标**: 获取电话联系方式
- **特点**:
  - 首轮介绍医院和加密声明
  - 第7轮开始尝试获取电话

### 4. 心理医院在线咨询
- **场景**: 心理健康咨询（失眠、焦虑等）
- **目标**: 获取电话或微信
- **特点**:
  - 首句包含医院介绍和联系方式
  - 优先获取电话，其次微信
  - 每8轮尝试一次获取联系方式

### 5. 儿童发育行为中心
- **场景**: 儿童多动症、注意力问题
- **目标**: 获取联系方式（电话/微信）

  - 第6轮开始尝试获取电话
  - 强调中医经络疗法

## 共同特征

### 对话流程
1. **首句规则**: 需要暖场、介绍医院/自己
2. **信息获取顺序**: 年龄 → 性别 → 症状表现 → 持续时间 → 就诊史
3. **轮次约束**: 在特定轮数内必须完成某些询问

### 禁忌约束
1. **禁用感谢用语**: 不使用"谢谢"、"感谢"等
2. **禁用解释性语句**: 不使用"这有助于..."、"了解到..."等
3. **禁止复述信息**: 用户提供信息后不复述
4. **禁止提前结束对话**: 未获取联系方式前不结束

### 获客策略
1. **留联顺序**: 通常先微信后电话，或先电话后微信
2. **时机**: 多在第6-9轮开始尝试获取联系方式
3. **理由**: 发送测评表、详细指导、案例分析等
4. **持续性**: 持续尝试直到获得或对话终止
